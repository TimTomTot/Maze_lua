Module Map

Модуль представляющий карту игрового мира

Позволяет получать лоступ к отдельным ячейкам на карте, а так же определять возможность пройти из одной ячейки в другую и видимость между ячейками

isWalkableBetween (srcX, srcY, distX, distY)
Проверка, возможно ли пройти между указанными точками

Parameters:

  • srcX X-координата точки отправления
  • srcY Y-координата точки отправления
  • distX X-координата точки назначения
  • distY Y-координата точки назначения

Returns:

  1. true если возможно пройти между точками, иначе false
  2. список объектов cell вдоль маршрута для прохода между точками отправления и назначения (без точки src)
isVisibleBetween (srcX, srcY, distX, distY)
Проверка, просматриваема ли точка dist из точки src

Parameters:

  • srcX X-координата точки отправления
  • srcY Y-координата точки отправления
  • distX X-координата точки назначения
  • distY Y-координата точки назначения

Returns:

  1. true если между точками есть прямая видимость, иначе false
  2. список объектов cell видимых вдоль луча зрения от точки src до точки dist (без точки src)
getNeighbors (x, y, diags)
Получить список всех соседних точек для заданой

Parameters:

  • x X-координата
  • y Y-координата
  • diags нужно ли рассматривать соседей по диагонали. Если нужно - должно быть установлено в true. По умолчанию false

Returns:

    список объектов cell, являющихся соседями заданой точки
checkRectangle (left, right, top, bottom)
Проверка, что прямоугольник полностью вписывается в границы карты

Parameters:

  • left Х-координата левого края прямоугольника
  • right Х-координата правого края прямоугольника
  • top Y-координата верхнего края прямоугольника
  • bottom Y-координата нижнего края прямоугольника

Returns:

    true если прямоугольник вписывается в размеры карты, иначе false
getCellList (left, right, top, bottom)
Получить список всех ячеек внутри прямоугольника Если размеры прямоугольника не заданы - то возвращается список всех ячеек на карте

Parameters:

  • left Х-координата левого края прямоугольника, по умолчанию 1
  • right Х-координата правого края прямоугольника, по умолчанию размер карты по горизонтали
  • top Y-координата верхнего края прямоугольника, по умолчанию 1
  • bottom Y-координата нижнего края прямоугольника, по умолчанию размер карты по вертикали

Returns:

    список объектов cell, попадающих внутрь заданного прямоугольника
