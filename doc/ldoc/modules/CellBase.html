Base:init (data)
Конструктор

Parameters:

  • data таблица со всеми инициализирующими значениями:
    • ID уникальный числовой идентификатор ячейки
    • x х-координата ячейки (необязательный параметр)
    • y у-координата ячейки (необязательный параметр)
    • name имя тайла
    • tiletype тип тайла, специфичный для некоторого набора - напремер все двери будут иметь одинаковый tiletype
    • tile тайл, в таком виде, как он должен парситься из строки с картой - одиночный символ
    • standmsg сообщение, которое будет отображаться, когда персонаж игрока будет заходить на данный тайл (необязательный параметр)
    • walkable является ли тайл проходимым
    • transparent является ли тайл прозрачным
    • explored исследован ли тайл (по умолчанию false)
    • shaded затенен ли тайл (по умолчанию true)
    • cancreature может ли на тайл вставать существо
    • canobject может ли на тайле находиться игровой предмет
